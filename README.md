# api_demo

> 一个开源的/无服务器的api测试工具(SPMS分支)

相关介绍和教程请移步[**官网**](https://adtl.gitee.io/pages/)

> SPMS分之,为spms架构定制分之

> 要求

* node 10

> 构建步骤

``` bash
# 
npm install

# 测试运行,热更新的
npm run dev

# 打包,当前平台
npm run build

# 运行unit和端到端测试
npm test


# lint all JS/Vue component files in `src/`
npm run lint

```

---


> 特色：

* 开源,免费,可修改
* 易分享（推荐使用版本控制工具分享如git，svn等）
* 无服务器依赖
* 结构化数据
* 版本控制
* Mock数据生成
* 自动化脚本测试/流水线测试(开发中)
